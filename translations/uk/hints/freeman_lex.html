Загальна ідея, яку я тобі пропоную використати, це створити сортовані списки з символів в одному регістрі, відмінних від пробілу, з обох вхідних рядків і порівняти їх.
З якого інструменту мені варто розпочати?
Щоб зробити всі літери в рядку в одному регістрі, використай рядкові методи str.lower() або str.upper(). 

a = "HeLlO"
a = a.lower()
print(a)

# output
# hello
Що робити далі? Потрібна підказка.
Щоб усунути пробіли, використай рядковий метод str.isalpha() для перевірки чи символ є літерою. 

a = "H"
print(a.isalpha())

# output
True
Я не розумію, як поєднати ці підказки.
Використай попередні дві підказки і списковий вираз, щоб вибрати з рядка у нижньому регістрі символи, що не є пробілом. Використай цей вираз як аргумент в функції sorted(). 

sorted(x for x in s.lower() if x.isalpha())
Як мені використати цей вираз для розв'язку місії?
Щоб писати цей вираз лише раз у функції і щоб зробити код функції чистішим, я пропоную тобі створити додаткову функцію для перетворення слова або фрази відповідно до цього виразу. Крім тго, це зробить простішим керування змінами, якщо тобі потрібно буде змінити алгоритм конвертування. 

def convert(s):

    return sorted(x for x in s.lower() if x.isalpha())
У головній функції просто викличи функцію convert для обох вхідних рядків, порівняй результати і поверни, чи рядки однакові.
То як має виглядати мій остаточний розв'язок?
Ось він: 

def verify_anagrams(a, b):
    
    return convert(a) == convert(b)
        
def convert(s):
            
    return sorted(x for x in s.lower() if x.isalpha())
Ти можеш використати lambda вираз (анонімну функцію) і функцію filter(), щоб зробити розв'язок більш компактним. 

verify_anagrams = lambda a, b: convert(a) == convert(b)

convert = lambda s: sorted(filter(lambda x: x.isalpha(), s.lower()))

Інший спосіб позбутися пробілів, це замінити їх порожнім рядком: 

def verify_anagrams(a,b):

    return sorted(a.lower().replace(' ',''))\
    == sorted(b.lower().replace(' ',''))
І ще інший спосіб - використати рядковий метод str.split(). 

def verify_anagrams(first_word, second_word):

    rs = lambda x: sorted("".join(x.lower().split()))
    
    return rs(first_word) == rs(second_word)
